# Customer Churn Analysis

## Objective
Analyze customer churn behavior to identify key factors influencing customer retention.

## Dataset
Telecom customer churn dataset containing demographic, service, and billing information.

## Tools Used
- Python
- pandas
- numpy
- matplotlib
- seaborn

## Key Insights
- Churn is higher among senior customers and customers with short tenure.
- Customers who churn tend to have higher average monthly charges.
- Customers with dependents show lower churn rates.
- Payment method and contract type influence billing and retention behavior.

## Business Implications
- Focus retention strategies on early-stage and high-value customers.
- Develop targeted retention initiatives for senior customers.
- Encourage long-term contracts and bundled services.
